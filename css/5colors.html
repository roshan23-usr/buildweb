colors

colors are difined by:

  1. color keyword

    2. defined by color name red, blue....

  2. RGB(x, y, z)

    3. defined by Red, Green, blue combination with given importance

  3. RGBA(w, x, y, z)

    4. defined by Red, Green, blue, Alpha=blur combination with given importance

  4. Hexa code

    5. defined by hexa code like #00ff00

  5. HSL(x, y%, z%)

    6. defined by Hue, Saturated, Light combination

  6. HSLA(w, x%, y%, z%)

    7. defined by Hue, Saturated, Light, Alpha=blur combination with given importance
